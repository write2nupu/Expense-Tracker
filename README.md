# Expense-Tracker

A modern fintech-style expense tracker iOS app built using SwiftUI and MVVM architecture.

## Features

- Add income & expenses
- Category-based transaction tracking
- Monthly financial summary
- Dark / Light mode support
- Smooth animations and micro-interactions
- Clean fintech-style UI
- Local data persistence

## Tech Stack

- SwiftUI
- MVVM Architecture
- CoreData (or UserDefaults)
- Charts Framework (for insights)

## Screens

- Auth (Sign In / Sign Up)
- Home Dashboard
- Transactions
- Insights
- Profile

## Setup Instructions

1. Clone the repo
2. Open in Xcode
3. Run on iOS Simulator (iOS 16+)
